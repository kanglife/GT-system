# 블록체인 과제시험: 재화 교환 서비스

## 목표

- Web2 기반의 포인트 재화를 개발합니다.
- 블록체인 기반의 토큰을 개발합니다. 이 토큰은 거버넌스 토큰이며 투표 기능을 제공해야합니다.
- 포인트는 고정 비율로 토큰과 교환할 수 있습니다.
- 모든 기능을 테스트할 수 있는 웹 어플리케이션을 개발합니다.

## 메인넷
